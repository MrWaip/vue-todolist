# vue-resume

# Запуск приложения

Нужно поднять контейнер с nginx, который будет доступен **8080** порту.
Он будет отдавать статику.

```
docker-compose up -d
```


## Project setup

```
pnpm install
```

### Compiles and hot-reloads for development

```
pnpm run serve
```

### Compiles and minifies for production

```
pnpm run build
```

### Lints and fixes files

```
pnpm run lint
```
